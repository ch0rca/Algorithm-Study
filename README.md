# 🧠 Algorithm Study Repository

알고리즘 문제 해결 능력 향상을 위해 다양한 문제를 풀이하고 기록하는 저장소입니다.  
꾸준한 학습과 복습을 통해 문제 해결 능력과 코드 구현 능력을 강화하는 것을 목표로 합니다.

---

## 🚀 Goals

- 다양한 알고리즘 및 자료구조 학습
- 문제 해결 접근 방식 개선
- 시간/공간 복잡도 최적화 능력 향상
- 꾸준한 코딩 습관 형성

---

## ✏️ How I Solve Problems

1. 문제 이해 및 조건 분석
2. 접근 방법 설계 (Brute-force, Greedy, DP 등)
3. 코드 구현
4. 시간/공간 복잡도 분석
5. 리팩토링 및 다른 풀이 방법 탐색

---

## 🛠️ Tech Stack

- Language: `Python` / `Java` 
- Tools: Git, GitHub

---

## 📈 Progress

- 하루 최소 1문제 이상 풀이를 목표
- 주기적으로 이전 문제를 복습

---

## 📝 Convention

- 파일명: `문제번호_문제이름`
  - ex) `1234_최단경로.py`
- 커밋 메시지:
  - `feat: 문제 풀이 추가`
  - `fix: 코드 수정`
  - `refactor: 리팩토링`

- 다양한 풀이 방법을 비교하며 더 나은 방법을 고민
